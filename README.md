# Overview

Notebooks is extension for Firefox that attempts to parse https://bactra.org/notebooks/ and notify user of new additions to the page.
The web page has an RSS feed, but it seems to be broken, or at least a few RSS readers extensions don't work with it.
It is thus just an RSS replacement for a single site.  
Of course, it would be easier to contact the web site owner and ask
him to correct the RSS feed, but I think this was a good opportunity
to learn Javascript.
